# Backend de Sistema de Inventario

Estructura base del backend para un sistema de inventario.

## Carpetas principales

- `src/config` - Configuración de base de datos y variables de entorno
- `src/controllers` - Controladores que reciben peticiones HTTP
- `src/routes` - Definición de rutas y endpoints
- `src/models` - Modelos de datos
- `src/services` - Lógica de negocio
- `src/middlewares` - Middlewares para auth y errores
- `src/utils` - Utilidades generales
- `tests` - Pruebas unitarias
- `docs` - Documentación de API

## Comandos

- `npm install`
- `npm run dev`
- `npm start`
